About NexusPad

It's a online tool that can be used to collect and organize important things.

Fundamentally it's not much different from a physical notebook, Rolodex or planner. When I created this tool, I wanted to have all my useful information conveniently gathered so I can find them when I need it.

How to use

Your data and privacy

An email is all you need to start using the tool. No other personal information is asked.
User input data is hosted and stored in secured database and Amazon Web Service servers. You own your data and it will never be shared or sold to any company or individual.
There is no analytics software on NexusPad domain so none of your browser information is passed to other companies. And there is no ads served to collect your information.
